# Satellite Property Valuation (Multimodal ML)

This project predicts property prices using satellite imagery and tabular housing data.

## Files
- data_fetcher.py : Downloads satellite images via Mapbox API
- preprocessing.ipynb : Data cleaning & EDA
- model_training.ipynb : CNN + tabular training & evaluation
